# employee-registeration-form-vuejs

## Project install
```bash
git pull https://github.com/ellaidevs/employee-registeration-form-vuejs.git
```

## Project setup
```
npm install
```

## Dependencies
```
- node >= v16.10.x
- npm >= 7.24.x
- firebase ^8.10.0
- lodash@core ^4.17.21
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
